## Vesting

**Vesting is a time unlocked ERC20 /vesting contract.**
This allows a payer to deposit a certain amount of tokens into a contract, but the reciever
can only withdraw 1/n tokens over the period of n days.

## Usage

### Build

```shell
$ forge build
```

### Test

```shell
$ forge test
```
